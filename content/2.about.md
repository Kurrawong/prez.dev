# About Prez

Prez was initially developed in ~2019 as VocPrez as a web API + templates-based tool to provide pleasant data display for vocabularies created according to the [SKOS model](https://www.w3.org/TR/skos-reference/).

It has since developed to be capable of providing views of almost any RDF / Knowledge Graph data: vocabularies, ontologies, spatial datasets, catalogues, anything! 

Prez is now made of two main components:

* [PrezAPI](/prezapi) - providing SHACL-based views of data in an RDF database
* [PrezUI](/prezui) - presentation of annotated RDF for JavaScript tools and web pages

There are also a number of support components, in particular:

* [KG Manifest](/kgmanifest) - a Knowledge Graph content orchestration tool
