A JS library for processing RDF data for use with Prez UI, implementing [RDF/JS](https://rdf.js.org/) and [N3.js](https://github.com/rdfjs/N3.js).

## Installing

:PackageManagerTabs{:data='{"npm": "npm install prez-lib", "pnpm": "pnpm add prez-lib", "yarn": "yarn add prez-lib"}'}

## Example Usage
Using the factory functions:

```typescript
import { literal, node } from "prez-lib";

const term = node({
    value: "https://example.com",
    label: literal("term"),
    links: [
        {
            value: "/"
        }
    ],
});
```

Using the store class:
```typescript
import { RDFStore } from "prez-lib";

const data = "<https://example.com/subject> <https://example.com/predicate> <https://example.com/object> .";
const store = new RDFStore();
store.load(data);
const item = store.getItem();
```
