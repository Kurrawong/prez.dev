Prez is a family of open source tools that is used to access, package, provice access to and present [RDF](https://en.wikipedia.org/wiki/Resource_Description_Framework) / [Knowledge Graph](https://en.wikipedia.org/wiki/Knowledge_graph) / [Semantic Web](https://en.wikipedia.org/wiki/Semantic_Web) data.

Prez is open source - see the [API repo](https://github.com/rdfLib/prez) and the [UI Repo](https://github.com/RDFLib/prez-ui/) - and it interacts with databases via the [SPARQL Protocol](https://www.w3.org/TR/sparql12-graph-store-protocol/) so you can use it with many storage tools. 

![Prez info flow](/img/prez-flow.svg){class="mx-auto"}

PrezAPI reads Knowledge Graph data using [SHACL](https://w3c.github.io/data-shapes/shacl12-core/) definitions which create profiles of the graph at different endpoints. It then adds in element labels and other information needed for easy API and front-end use and delivers data via Python's [FastAPI](https://fastapi.tiangolo.com/). PrezUI implements JavaScript data presentatation libraries and a Vue.js-based website for web pages.
